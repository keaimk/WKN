# WKN

## 🌎 프로젝트 소개
"WKN"은 해외 거주 지역을 기반으로 커뮤니티를 형성하고, 해외 생활에 필요한 다양한 정보를 자유롭게 공유하고 소통할 수 있는 플랫폼입니다.
